# 🎧 Spotify Clone (Frontend)

A visually rich and responsive **Spotify-inspired music player UI** built using **HTML, CSS, and JavaScript**.
This project replicates the core layout and user experience of modern music streaming platforms, including sidebar navigation, playlists, and a custom audio player.

---

## 🚀 Features

* 🎵 Interactive music player (Play / Pause / Next / Previous)
* 📃 Recently played songs section
* 📂 Sidebar navigation (Home, Search, Library)
* ❤️ Like songs functionality
* 🔊 Volume & progress bar controls
* 🖼 Dynamic album artwork display
* 🎧 Playlist cards (Chill, Party, Rap, etc.)
* 📱 Fully responsive modern UI

## ⚙️ How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/shivamkumar129/Spotify.git
```

2. Open the project folder:

```bash
cd Spotify
```

3. Run the app:

* Simply open `index.html` in your browser

## 🎮 Usage

* Click on any song to start playing 🎶
* Use the bottom player controls to manage playback
* Adjust volume and seek using sliders
* Browse playlists from the home section
